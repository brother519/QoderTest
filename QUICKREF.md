# 🎮 坦克大战 - 快速参考

## 🚀 快速启动
```bash
./start.sh
# 或
npm install && npm run dev
```

## 🎯 游戏操作

### 单人模式
| 操作 | 按键 |
|------|------|
| 移动 | WASD 或 方向键 |
| 射击 | J 或 空格 |
| 暂停 | P 或 ESC |

### 双人模式
| 玩家 | 移动 | 射击 |
|------|------|------|
| P1(黄) | WASD | J |
| P2(绿) | 方向键 | 1 |

## ✅ 游戏规则

### 胜利条件
- ✓ 消灭20辆敌方坦克
- ✓ 保护基地不被摧毁

### 失败条件
- ✗ 基地被摧毁
- ✗ 生命值耗尽

## 💡 游戏技巧
1. **优先保护基地** - 基地被毁立即失败
2. **利用掩体** - 使用墙壁作为掩护
3. **控制节奏** - 不让太多敌人同时出现
4. **善用道具** - 在关键时刻使用道具
5. **预判射击** - 提前预判敌人位置

## 🐛 问题排查

### 游戏无法启动
1. 检查浏览器是否支持ES6
2. 确保通过HTTP服务器访问
3. 查看浏览器控制台错误

### 操作无响应
1. 点击游戏画面获取焦点
2. 检查按键是否被占用
3. 尝试刷新页面

### 画面卡顿
1. 关闭其他占用资源的程序
2. 更新浏览器到最新版本
3. 降低浏览器缩放比例
