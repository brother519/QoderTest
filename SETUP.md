# 项目安装和运行指南

## 环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

## 安装步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 已知功能特性

### 数据持久化
- 数据存储在内存中，页面刷新后会重置
- 这是模拟API的预期行为，生产环境应连接真实后端

### 本地化
- 全中文界面
- 日期选择器支持中文格式
- 时间显示使用中文格式

### 性能优化
- 搜索防抖：300ms延迟减少API调用
- 组件懒加载：模态框按需渲染
