# 贪吃蛇游戏

一个使用 Go 语言实现的终端版本贪吃蛇游戏。

## 功能特点

- 经典贪吃蛇游戏玩法
- 终端界面显示
- 方向键控制蛇的移动
- 实时计分系统
- 游戏结束后可重新开始

## 游戏规则

- 使用方向键（↑↓←→）控制蛇的移动方向
- 吃到食物（`*`）后蛇会变长，得分增加10分
- 撞到墙壁（`#`）或自己的身体会导致游戏结束
- 蛇头显示为 `@`，蛇身显示为 `o`

## 运行方法

### 方式一：直接运行编译好的可执行文件

```bash
./snake
```

### 方式二：从源码编译运行

```bash
# 下载依赖
go mod tidy

# 编译
go build -o snake snake_game.go

# 运行
./snake
```

## 操作说明

- **方向键**：控制蛇的移动方向
- **Q 键**：退出游戏
- **R 键**：游戏结束后重新开始

## 技术实现

- 使用 Go 语言开发
- 使用 `github.com/eiannone/keyboard` 库处理键盘输入
- 使用定时器控制游戏刷新速度

## 文件说明

- `snake_game.go` - 游戏主程序
- `snake_game.py` - Python版本（需要Python环境和curses库）
- `go.mod` - Go模块依赖配置
- `snake` - 编译后的可执行文件
