# 拳皇游戏 (KOF) 项目

## 项目概述

这是一个基于Unity的2D格斗游戏，实现了拳皇系列游戏的核心机制，包括角色选择、战斗系统、技能系统等。

## 项目结构

```
KOF-Game/
├── Assets/
│   ├── Scripts/          # 游戏脚本
│   │   ├── Core/         # 核心系统
│   │   ├── Characters/   # 角色相关
│   │   ├── Combat/       # 战斗系统
│   │   ├── UI/           # 用户界面
│   │   └── Data/         # 数据文件
│   ├── Scenes/           # 游戏场景
│   ├── Sprites/          # 图像资源
│   ├── Audio/            # 音频资源
│   └── Prefabs/          # 预制件
```

## 核心功能

1. **角色系统** - 多个可玩角色，每个角色有独特技能
2. **战斗系统** - 2D格斗机制，包含攻击、防御、闪避
3. **技能系统** - 特殊技能和必杀技
4. **游戏模式** - 单人模式、双人对战模式
5. **UI界面** - 完整的用户界面系统

## 技术实现

项目使用Unity 2D引擎开发，主要技术栈包括：
- C# 脚本编程
- Unity 2D工具集
- 物理系统
- 动画系统
- UI系统

## 快速开始

1. 使用Unity打开项目
2. 配置输入管理器（参考实现方案文档）
3. 运行MainMenu场景开始游戏

## 游戏操作

### 玩家1 (键盘)
- 移动: A/D 或 方向键左右
- 跳跃: W 或 上方向键
- 攻击: J
- 技能1: K
- 技能2: L
- 技能3: I

### 玩家2 (键盘)
- 移动: 方向键
- 跳跃: 小键盘8
- 攻击: 小键盘5
- 技能1: 小键盘4
- 技能2: 小键盘6
- 技能3: 小键盘7
