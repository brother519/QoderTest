# 坦克大战游戏

一个使用 Next.js、TypeScript、Tailwind CSS 和 Canvas 实现的经典坦克大战游戏。

## 游戏特性

- 玩家坦克控制（WASD或方向键移动，空格键射击）
- 智能敌方坦克AI（自动移动和射击）
- 砖块和钢铁障碍物
- 生命值系统
- 计分系统
- 游戏结束和获胜条件

## 游戏操作

- **移动**: W/A/S/D 键或方向键
- **射击**: 空格键
- **开始/重启游戏**: 点击按钮

## 游戏目标

- 消灭所有敌方坦克获胜
- 避免被敌方坦克和子弹击中
- 利用障碍物进行掩护

## 技术栈

- **React**: 使用Next.js 14
- **TypeScript**: 类型安全
- **Tailwind CSS**: 样式设计
- **HTML5 Canvas**: 游戏渲染

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 运行开发服务器：
```bash
npm run dev
```

3. 在浏览器中访问 http://localhost:3000
