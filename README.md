# 상품 판매 시스템

React + Vite 기반으로 구축된 현대적인 상품 판매 페이지 시스템으로, 완전한 전자상거래 기능이 통합되어 있습니다.

## 🚀 기능 특징

### 핵심 기능
- **상품 전시** - 상품 상세 정보, 이미지 캐러셀, 사양 선택
- **장바구니 관리** - 상품 추가, 수량 조절, 가격 계산
- **사용자 시스템** - 사용자 정보, 빠른 작업, 개인화된 경험
- **댓글 시스템** - 사용자 평가, 평점 통계, 필터링 및 정렬
- **검색 기능** - 키워드 검색, 필터, 정렬
- **결제 시스템** - 다양한 결제 방식, 보안 인증
- **주소 관리** - 배송 주소, 기본 주소 설정

### 기술 특징
- **반응형 디자인** - 모바일, 태블릿, 데스크톱 지원
- **상태 관리** - Zustand를 사용한 전역 상태 관리
- **데이터 영속성** - 장바구니 및 사용자 정보 로컬 스토리지
- **컴포넌트 아키텍처** - 재사용 가능한 React 컴포넌트
- **현대적인 UI** - Tailwind CSS 스타일 프레임워크

## 🛠️ 기술 스택

- **프론트엔드 프레임워크**: React 18
- **빌드 도구**: Vite
- **상태 관리**: Zustand
- **라우팅**: React Router DOM
- **스타일링**: Tailwind CSS
- **아이콘**: Lucide React
- **개발 언어**: JavaScript/JSX

## 📁 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── ShoppingCart/    # 장바구니 컴포넌트
│   ├── ProductDisplay/  # 상품 전시 컴포넌트
│   ├── CommentList/     # 댓글 목록 컴포넌트
│   ├── UserCard/        # 사용자 정보 카드
│   ├── Payment/         # 결제 컴포넌트
│   ├── AddressSelector/ # 주소 선택기
│   └── SearchResultList/# 검색 결과 목록
├── store/               # 상태 관리
│   ├── productStore.js  # 상품 상태
│   ├── cartStore.js     # 장바구니 상태
│   ├── userStore.js     # 사용자 상태
│   ├── addressStore.js  # 주소 상태
│   ├── paymentStore.js  # 결제 상태
│   └── commentStore.js  # 댓글 상태
├── pages/               # 페이지 컴포넌트
│   └── HomePage.jsx     # 홈페이지
├── data/                # 모의 데이터
│   └── mockData.js      # Mock 데이터
├── types/               # 타입 정의
│   └── index.js         # 데이터 모델
├── utils/               # 유틸리티 함수
├── hooks/               # 커스텀 Hooks
└── api/                 # API 서비스 레이어
```

## 🎯 핵심 컴포넌트 설명

### ShoppingCart (장바구니 컴포넌트)
- 사이드바 형식의 장바구니 인터페이스
- 상품 수량 조절
- 실시간 가격 계산
- 상품 사양 선택 지원
- 로컬 스토리지 데이터 영속성

### ProductDisplay (상품 전시 컴포넌트)
- 상품 이미지 캐러셀
- 상세 정보 표시
- 사양 선택기
- 재고 표시
- 평점 및 태그

### CommentList (댓글 목록 컴포넌트)
- 댓글 표시 및 필터링
- 평점 통계
- 정렬 기능
- 페이지네이션 로딩
- 좋아요 상호작용

### UserCard (사용자 정보 카드)
- 사용자 아바타 및 기본 정보
- 회원 등급 표시
- 빠른 작업 메뉴
- 통계 데이터 표시
