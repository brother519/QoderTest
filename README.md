# 统一业务平台

## 项目概述

这是一个经过架构重构的统一业务平台，整合了产品销售和任务管理两大核心业务模块。项目采用现代化的技术栈和规范化的架构设计，确保代码的可维护性、可扩展性和性能表现。

## 技术栈

### 核心技术
- **React 18** - 前端框架
- **TypeScript** - 类型安全的JavaScript超集
- **Material-UI (MUI)** - 企业级UI组件库
- **Zustand** - 轻量级状态管理
- **React Router v6** - 路由管理
- **Vite** - 现代化构建工具

### 开发工具
- **ESLint** - 代码质量检查
- **TypeScript** - 静态类型检查
- **Vitest** - 单元测试框架
- **Testing Library** - React组件测试

## 项目结构

```
src/
├── app/                    # 应用程序入口
│   ├── App.tsx
│   └── main.tsx
├── features/               # 业务功能模块
│   ├── product/            # 产品销售模块
│   │   ├── components/     # 产品相关组件
│   │   ├── pages/          # 产品页面
│   │   └── store/          # 产品状态管理
│   └── task/               # 任务管理模块
│       ├── components/     # 任务相关组件
│       ├── pages/          # 任务页面
│       └── store/          # 任务状态管理
├── shared/                 # 共享资源
│   ├── components/         # 通用组件
│   ├── types/              # 类型定义
│   ├── utils/              # 工具函数
│   └── hooks/              # 自定义Hook
├── infrastructure/         # 基础设施
│   ├── theme/              # Material-UI主题
│   └── router/             # 路由配置
└── assets/                 # 静态资源
    └── styles/             # 样式文件
```

## 功能模块

### 🛍️ 产品销售模块
- **产品展示** - 商品列表和详情页面
- **购物车管理** - 添加、删除、修改商品数量
- **商品搜索** - 支持关键词搜索和筛选
- **商品变体** - 支持多规格商品选择
- **价格计算** - 实时价格更新和总价计算

### 📝 任务管理模块
- **任务创建** - 支持创建带有优先级、分类的任务
- **任务列表** - 清晰的任务展示和状态管理
- **任务筛选** - 按状态、优先级、分类筛选任务
- **任务统计** - 实时统计任务完成情况
- **状态流转** - 支持任务状态的灵活切换

## 架构设计特点

### 🏗️ 分层架构
- **展示层** - React组件负责UI渲染
- **业务逻辑层** - Zustand Store管理业务状态
- **基础设施层** - 路由、主题、工具函数

### 🔧 模块化设计
- **业务模块分离** - 产品和任务模块完全独立
- **共享资源复用** - 通用组件和工具函数统一管理
- **类型安全** - 完整的TypeScript类型定义

### ⚡ 性能优化
- **代码分割** - 按模块懒加载，减少初始包大小
- **Bundle优化** - 合理的依赖分包策略
- **组件优化** - 使用React.memo等优化技术

### 🧪 测试策略
- **单元测试** - 组件和函数级别测试
- **集成测试** - Store和业务逻辑测试
- **类型检查** - TypeScript静态类型验证

## 开发规范

### 代码风格
- 使用ESLint进行代码质量检查
- 统一的命名规范和文件组织
- TypeScript严格模式确保类型安全

### 组件设计
- 函数式组件优先
- Props接口严格类型定义
- 合理的组件职责划分

### 状态管理
- Zustand统一状态管理
- Store按业务模块分离
- 支持数据持久化

- **自定义服务器** - Nginx/Apache等
