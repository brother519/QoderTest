# 连连看游戏

一个基于Web浏览器的经典连连看游戏,支持可配置的棋盘规格和多关卡挑战。

## 功能特性

### 核心功能
- ✅ **可配置棋盘规格**: 支持6×6、8×8、10×10等不同尺寸的棋盘
- ✅ **多关卡系统**: 5个难度递增的关卡
- ✅ **路径搜索算法**: 使用BFS算法实现最多2次转折的路径查找
- ✅ **计分系统**: 基础分数 + 连击奖励 + 时间奖励
- ✅ **倒计时系统**: 每个关卡有时间限制,增加挑战性
- ✅ **关卡进度保存**: 自动保存解锁的关卡进度

### 游戏规则
- 点击两个相同图案的方块进行消除
- 两个方块之间必须存在有效连接路径
- 路径不能穿过其他方块
- 路径转折次数不超过2次
- 消除所有方块即可过关

### 计分规则
- 基础消除: 每对100分
- 连击奖励:
  - 第1次: 1倍
  - 第2次: 1.2倍
  - 第3次: 1.5倍
  - 第4次及以上: 2倍
- 时间奖励: 剩余秒数 × 10

### 关卡配置

| 关卡 | 棋盘规格 | 图案种类 | 时间限制 | 说明 |
|------|---------|---------|---------|------|
| 1 | 6×6 | 4 | 120秒 | 入门关卡 |
| 2 | 8×8 | 6 | 150秒 | 标准难度 |
| 3 | 8×8 | 8 | 120秒 | 时间压力 |
| 4 | 10×10 | 10 | 180秒 | 大棋盘 |
| 5 | 10×10 | 12 | 150秒 | 高难度 |

## 技术架构

### 项目结构
```
QoderTest/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
└── js/
    ├── models.js       # 数据模型
    ├── levelManager.js # 关卡管理器
    ├── pathAlgorithm.js# 路径搜索算法
    ├── boardManager.js # 棋盘管理器
    ├── scoreManager.js # 计分管理器
    ├── gameController.js# 游戏核心控制器
    ├── ui.js          # UI组件管理
    └── main.js        # 主入口
```

### 核心模块

1. **数据模型 (models.js)**
   - GameState: 游戏状态枚举
   - Tile: 方块类
   - Board: 棋盘类
   - LevelConfig: 关卡配置类
   - GameStateData: 游戏状态数据类

2. **关卡管理器 (levelManager.js)**
   - 加载和管理5个关卡配置
   - 追踪关卡进度
   - 本地存储支持

3. **路径算法引擎 (pathAlgorithm.js)**
   - BFS广度优先搜索
   - 支持虚拟边界扩展
   - 最多2次转折限制

4. **棋盘管理器 (boardManager.js)**
   - 动态生成不同规格的棋盘
   - 方块选择和消除逻辑
   - 确保图案成对出现

5. **计分管理器 (scoreManager.js)**
   - 基础分数计算
   - 连击倍数管理
   - 时间奖励计算

6. **游戏核心控制器 (gameController.js)**
   - 协调各模块交互
   - 管理游戏生命周期
   - 处理游戏流程转换

7. **UI组件管理 (ui.js)**
   - Canvas渲染棋盘
   - 响应式布局适配
   - 动画效果展示

## 如何运行

### 方法1: 直接打开
直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 方法2: 使用本地服务器
```bash
# 使用Python启动简单HTTP服务器
cd /data/workspace/QoderTest
python3 -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000
```

### 方法3: 使用Node.js
```bash
# 安装http-server
npm install -g http-server

# 启动服务器
cd /data/workspace/QoderTest
http-server -p 8000
```

## 操作说明

1. **开始游戏**: 点击"开始游戏"进入第一关
2. **选择关卡**: 点击"选择关卡"挑战已解锁的关卡
3. **游戏操作**:
   - 点击第一个方块选中
   - 点击第二个相同图案的方块尝试消除
   - 如果路径有效,方块将被消除
4. **游戏控制**:
   - 暂停: 暂停游戏并冻结倒计时
   - 重新开始: 重置当前关卡
   - 退出: 返回主菜单
