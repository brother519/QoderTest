# 坦克大战游戏

经典坦克大战游戏 - 基于Phaser 3游戏引擎开发

## 功能特性

- ✅ 单人模式:玩家对抗AI敌人
- ✅ 双人模式:本地双人协作
- ✅ 多种地形:可破坏砖墙、不可破坏钢墙
- ✅ 基地保护:保护基地不被摧毁
- ✅ 敌人AI:自动移动和射击
- ✅ 生命系统:3条生命,被击中重生
- ✅ 分数系统:击毁敌人获得分数
- ✅ HUD显示:实时显示生命、分数、剩余敌人

## 技术栈

- **游戏引擎**: Phaser 3.60+
- **开发语言**: TypeScript 5.x
- **构建工具**: Vite 5.x
- **物理引擎**: Phaser Arcade Physics

## 安装依赖

```bash
npm install
```

## 运行游戏

开发模式(带热更新):
```bash
npm run dev
```

构建生产版本:
```bash
npm run build
```

预览生产版本:
```bash
npm run preview
```

## 游戏操作

### 玩家1 (蓝色坦克)
- **移动**: 方向键 ↑↓←→
- **射击**: 空格键

### 玩家2 (绿色坦克 - 双人模式)
- **移动**: WASD键
- **射击**: J键

### 通用操作
- **暂停/返回菜单**: ESC键

## 游戏规则

1. 消灭所有敌人坦克获得胜利
2. 保护基地(黄色方块)不被摧毁
3. 玩家有3条生命,被击中后在出生点重生
4. 击毁每个敌人获得100分
5. 基地被摧毁或生命耗尽游戏结束

## 项目结构

```
src/
├── entities/          # 游戏实体(坦克、子弹、基地等)
├── scenes/            # 游戏场景(菜单、游戏、结束等)
├── terrain/           # 地形系统(砖墙、钢墙)
├── ai/                # AI控制器
├── managers/          # 游戏管理器(输入、碰撞、状态)
├── ui/                # UI组件(HUD)
├── types/             # TypeScript类型定义
└── utils/             # 工具函数和常量
```
